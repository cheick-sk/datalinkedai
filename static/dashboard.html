💼 LinkedIn — Publication Automatique

Configure ton token LinkedIn pour que les posts soient publiés automatiquement sur ton profil après validation. Sans ça, les posts s'affichent dans la validation pour copier-coller manuellement.

📖 Comment obtenir ton token LinkedIn ?
1. Va sur https://www.linkedin.com/developers/apps → Create App 2. Permissions requises : w_member_social (pour publier) 3. Dans "Auth" → génère un Access Token via OAuth 2.0 4. Ton Person ID : va sur ton profil LinkedIn → inspecte l'URL API ou utilise : GET https://api.linkedin.com/v2/me (avec le token) → le champ "id" est ton Person ID 5. Colle le token + Person ID ci-dessus → Sauvegarder 6. Active LINKEDIN_AUTO_POST=true dans Railway pour la publication auto ✅ Posts publiés automatiquement après validation !

🔔 Notifications — WhatsApp · SMS · Email

Le Copilot t'envoie une notification dès qu'il y a des éléments à valider. Tu cliques ✅ Valider ou ❌ Rejeter directement depuis ton téléphone — sans ouvrir le dashboard.

📱 Comment configurer WhatsApp / SMS (Twilio — gratuit)
1. Compte Twilio (gratuit) → twilio.com → Console → Account SID + Auth Token
2. WhatsApp Sandbox → Console → Messaging → Try it out → WhatsApp → Envoie le code depuis ton téléphone
3. Variables à configurer :
TWILIO_ACCOUNT_SID=ACxxx
TWILIO_AUTH_TOKEN=xxx
NOTIF_WHATSAPP=+33612345678 (ton numéro)
NOTIF_PHONE=+33612345678 (pour SMS)
NOTIF_CHANNELS=email,whatsapp,sms

Comment ça marche :
① Copilot scrape → analyse → génère CV + post
② Tu reçois un email/WhatsApp/SMS avec les éléments
③ Tu cliques ✅ ou ❌ depuis ton téléphone
④ L'envoi (email recruteur, post LinkedIn) se fait automatiquement
