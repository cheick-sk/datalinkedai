🗄️ Supabase — Étapes

1. Aller sur supabase.com → New project 2. Nom : "datalinkedai" | Region : West EU (Paris) 3. Mot de passe fort → Create project (2 min) 4. Settings → Database → Connection String → URI 5. Copier l'URL : postgresql://postgres:[PASS]@db.[REF].supabase.co:5432/postgres 6. Remplacer [PASS] par ton mot de passe 7. Coller dans Railway : DATABASE_URL = cette URL 8. Redémarrer le service → vérifier /health ✅ Données persistantes pour toujours !

🕷️ Apify — Étapes

1. Aller sur apify.com → Sign up gratuit 2. Settings → Integrations → API token 3. Copier ton token : apify_api_xxxxx... 4. Coller dans Railway : APIFY_TOKEN = ce token 5. Redémarrer le service 6. Tester : POST /api/copilot/run/sync 7. Vérifier "scraping":"apify" dans la réponse ✅ Vraies offres LinkedIn chaque matin à 8h ! Note: Plan gratuit = 5$/mois de crédits. 1 run copilot ≈ 0.05$ → 100 runs/mois gratuits.
